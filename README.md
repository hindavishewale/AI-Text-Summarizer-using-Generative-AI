# AI-Text-Summarizer-using-Generative-AI
AI-powered text summarizer that converts long text into short, clear summaries using generative AI.

This project is a simple AI text summarizer that converts long text into short and clear summaries using a generative AI model.  
Users can paste any paragraph and get a concise summary instantly.

## 🚀 Features
- Summarizes long text into short output  
- Simple and easy to use  
- Built using Generative AI  
- Implemented in Google Colab  
- Quick and clean results  

## 🛠️ Technologies Used
- Python  
- Generative AI  
- Natural Language Processing (NLP)  
- Google Colab  

## ▶️ How to Use
1. Open the Google Colab link below  
2. Run all cells  
3. Enter your text  
4. Get the summarized output  

## 🔗 Project Link
Google Colab Notebook:

## 🎯 Objective
To build a simple AI tool that helps users quickly understand long text by generating short summaries.
